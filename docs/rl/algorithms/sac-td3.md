# SAC / TD3（概要）

**off-policy** の連続制御アルゴリズムです。yuuki-lab 本線では未使用ですが、将来の比較候補として概要だけ記載します。

## SAC（Soft Actor-Critic）

| 項目 | 内容 |
|------|------|
| 思想 | 報酬最大化 + エントロピー最大化（最大エントロピー RL） |
| データ | off-policy（リプレイバッファ） |
| 長所 | サンプル効率が良いことが多い |
| 短所 | 実装・チューニングが PPO より重め |

## TD3（Twin Delayed DDPG）

| 項目 | 内容 |
|------|------|
| 思想 | Actor-Critic の off-policy 版。Q の過大評価を抑える工夫 |
| 長所 | 連続制御で実績あり |
| 短所 | ハイパラに敏感なことがある |

## PPO との選び方（ロボット歩行）

| 状況 | 候補 |
|------|------|
| シミュが速く on-policy で回せる | **PPO**（現状） |
| 実機でサンプルが極端に貴重 | SAC 等 off-policy を検討 |
| まず安定にベースライン | PPO が無難 |
